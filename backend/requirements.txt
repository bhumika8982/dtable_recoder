# NOTE: install torch + torchaudio matching your CUDA/CPU setup BEFORE these.
#   Tested-good CPU build (Windows / Python 3.11):
#     pip install torch==2.8.0 torchaudio==2.8.0 --index-url https://download.pytorch.org/whl/cpu
#   Version constraints:
#     * torchaudio must be <2.9 -- 2.9 removed `torchaudio.AudioMetaData`, which
#       pyannote.audio 3.3.2 still imports (ImportError otherwise).
#     * torch must be >=2.3 to support NumPy 2.x (we install numpy 2.x via pandas etc).
#     * 2.8.0 is the newest pair satisfying both.
#   Windows: torch needs the Microsoft Visual C++ Redistributable >=14.40 (VS 2015-2022).
#       An older redist (e.g. 14.29) loads but fails with "c10.dll ... DLL initialization
#       routine failed" (WinError 1114). Fix: install vc_redist.x64.exe from aka.ms/vs/17/release.
# Pin `pyannote.audio` to match `whisperx` dependency (3.3.2 required by whisperx 3.3.1)
# NOTE: whisperx >=3.3.1 pulls faster-whisper >=1.1.0, which relaxes the `av` pin to
# `av>=11` so pip can use the prebuilt av wheel (av 11.* has no Windows wheel and would
# otherwise require the MSVC C++ Build Tools to compile from source).
